# Instrucciones
## para ocupar jekyll exitosamente con este sitio (y con cualquier otro en realidad)

Antes de actualizar estos comandos hay que estar en la carpeta (un poco obvio)

Luego, para actualizar los paquetes (o GEM's ) de ruby hay que escribir `bundle update` en el terminal.

Y para echar a andar el sitio con las dependencias correctas, hay que escribir `bundle exec jekyll serve`.
